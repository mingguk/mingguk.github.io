자바스크립트에서 함수가 실행될 때마다 새로운 Execution context를 생성하고 각 Execution context 는 variable Environmnet, Outer Environment 그리고 this 를 생성한다. this 는 함수의 위치, 호출되는 방식에 따라 특정 객체를 가르키게 된다.
